一、概述

ArrayList是一个动态数组,特点是

  1. 线程不安全
  2. 允许元素为null
    实现了RandomAccess接口拥有随机快速访问的能力

当然因为底层是数组，所以也拥有数组的优缺点，连续的内存，读写时间效率高，空间效率低
因为是数组，所以扩容操作也是性能消耗大的地方。主要关心的也是扩容操作

二、构造方法

构造方法走完之后，数组的size和elementData就有了
构造方法中的一些工具方法很值得注意一下
